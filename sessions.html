📋 How to Switch the Live Version — Command Line

The live site is whatever is on the main branch. To switch, merge your desired version into main and push:

# Example: switch to the Peach version
git checkout main
git merge version-2-peach
git push

# Switch to Lavender:
git checkout main
git merge version-3-lavender
git push

# Switch back to Floral at any time:
git checkout main
git merge version-1-floral
git push

⚡ Netlify auto-detects every push and redeploys in ~30 seconds. No manual steps needed.

🖥 How to Switch Using GitHub Desktop

If you prefer a visual interface over the command line, GitHub Desktop makes switching versions easy:

Step-by-Step

  1. Open GitHub Desktop → Click "Add" → "Add Existing Repository" → select the our day folder
  2. In the top bar, make sure you're on the "main" branch (Current Branch dropdown)
  3. Click Branch → Merge into Current Branch (or Ctrl+Shift+M)
  4. Select the version you want: version-2-peach or version-3-lavender
  5. Click "Merge" — the version's files now replace main
  6. Click "Push origin" (or Ctrl+P) to upload to GitHub
  7. ⏱ Wait ~30 seconds — Netlify auto-redeploys your new live version!

Visual Walkthrough

① Current Branch:
Top bar → dropdown → select main
② Merge:
Branch menu → Merge into Current Branch → pick the version
③ Push:
Click "Push origin" button in the top bar (or Ctrl+P)
④ Done!
Netlify redeploys automatically. Check your live URL.

💡 Note: If merge conflicts happen (unlikely since each version is self-contained), GitHub Desktop will show them — just accept the incoming version's changes.

🌐 Switch via Netlify Dashboard (No Code)
  1. Go to app.netlify.com → select your ghazi-bayan site
  2. Click the "Deploys" tab in the left sidebar
  3. You'll see all branch deploys listed — each with a timestamp and preview link
  4. Hover over the deploy you want live → click "Publish deploy"
  5. ✅ Done! Instant switch — the new version is now live at ghazi-bayan.netlify.app

💡 This is the fastest method — no git commands, no app needed. Just click and it's live.
